API Reference

Endpoint Parameters Description
/api/get/int:sid
  • sid - user session ID
 Retrieves a subset of sensor data as JSON serialised file based on user session ID
/api/get/int:start/int:end/int:sid
  • start - start time of data subset as UNIX timestamp in milliseconds
  • end - end time of data subset as UNIX timestamp in in milliseconds
  • sid - user session ID
 Retrieves a subset of sensor data as JSON serialised file based on provided user session ID and from/to timestamps
/api/count/int:sid
  • sid - user session ID
 Returns the number of sensor samples collected over a session
/api/countAll None Returns the total number of sensor samples in the cloud storage
/api/delete/int:sid
  • sid - user session ID
 Deletes the sensor samples that belong to user session
/api/deleteAll None Deletes the entire dataset
/api/validate/int:sid
  • sid - user session ID
 Checks the consistency of subset of data that belongs to a session
/api/getSessions None Returns all sessions
/api/getLast None Returns the contents of the most recent sensor batch
